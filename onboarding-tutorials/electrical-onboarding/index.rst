Welcome new Members
*******************
Please complete required safety training, and install the following required software on your computer.

Required Softwares
******************

git
===
Git is a version control system, install it from `git website <https://git-scm.com/downloads>`_  or your package manager

KiCAD
=====
KiCAD is the PCB design software. We use KiCAD version 8, install KiCAD from `KiCAD website <https://www.kicad.org/download/>`_ or your package manager

MPLab and XC compilers
======================
MPLab is an IDE(Integrated Development Environment) for Microchip PIC microcontrollers, XC8 and XC16 are the compilers. Use the following links to download them, `MPLAB X IDE <https://www.microchip.com/en-us/tools-resources/develop/mplab-x-ide>`_ and the `XC8 and XC16 compilers <https://www.microchip.com/en-us/tools-resources/develop/mplab-xc-compilers>`_

omnibus
=======
`Refer to omnibus GitHub repositroy README.md <https://github.com/waterloo-rocketry/omnibus>`_

LLVM clang-format
=================
:doc:`Refer to clang-format page </onboarding-tutorials/firmware-how-to/clang-format>`

Optional Softwares
******************
These softwares are less commonly used(usually just 2-3 projects using it), so you only have to install them when working on a project needs it.

STM32CubeIDE
============
STM32CubeIde is an IDE(Integrated Development Environment) for STMicroelectronics STM32 series micro-controllers, download it from `STM32CubeIDE page <https://www.st.com/en/development-tools/stm32cubeide.html>`_

Arduino IDE
===========
Arduino is an IDE for Arduinos, download it from `Arduino IDE page <https://www.arduino.cc/en/software>`_ or from system package manager.
